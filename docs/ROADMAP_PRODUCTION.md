## 1. Executive Summary

**tssAI** — AI-приложение для спортсменов на выносливость (питание, сон, тренировки, AI-чат). Проект имеет два окружения: **Dev** (dev.tsspro.tech) и **Production** (tsspro.tech). Дорожная карта описывает этапы от текущего альфа-состояния до стабильной эксплуатации.

**Текущий статус:**
- ✅ Dev-сервер: 209.38.17.171
- ✅ Prod-сервер: 167.71.74.220
- ✅ CI/CD: GitHub Actions → GHCR
- ✅ Frontend unit-тесты: 6/6 passed
- ⚠️ Backend unit-тесты: требуют PostgreSQL и pytest
- ⚠️ E2E-тесты: Maestro (ручной запуск)

## 4. Команды быстрого доступа

| Действие | Команда |
|----------|---------|
| Локальный запуск | `make run` |
| Деплой на Dev | `make deploy-dev` |
| Деплой на Prod | `make deploy` |
| Деплой только backend | `make deploy-backend-no-push` или `BUILD_SERVICES=backend` |
| Деплой из CI (без сборки на сервере) | `make deploy-no-push USE_CI_IMAGES=1 CI_REGISTRY_OWNER=<owner> CI_IMAGE_TAG=latest` |
| Откат | `make deploy-no-push CI_IMAGE_TAG=<previous-version>` |
| Восстановление БД из S3 | `./deploy/restore-from-s3.sh latest` |
