# i18n — доработки после добавления локалей

Список задач для доведения мультиязычности после коммита `feat(i18n): add Thai and main locales`.

---

## Обязательно

- [x] **Полные переводы UI для de, fr, es, it, pt, th**  
  Полные объекты переводов вынесены в `frontend/src/i18n/locales/` (de, fr, es, it, pt, th) и подключены в `translations.ts`.

- [x] **Тесты: NavigationContainer**  
  В тестах `DashboardScreen` и `ChatScreen` рендер обёрнут в `NavigationContainer`; для кнопок чата используется `getByLabelText` (accessibilityLabel).

- [ ] **Проверка сценариев**  
  Прогнать вручную по чек-листу ниже.

---

## Чек-лист ручной проверки

Перед релизом прогнать вручную:

- [ ] **Смена языка в профиле** — выбор языка сохраняется, UI перерисовывается на выбранный язык.
- [ ] **Смена языка в меню дашборда** — то же поведение, выбор сохраняется.
- [ ] **Сохранение в профиле** — PATCH с `locale` уходит на бэкенд, после перезапуска приложения выбранный язык остаётся.
- [ ] **Заголовок X-App-Language** — в сетевых запросах (DevTools → Network) заголовок соответствует выбранной локали (ru, en, de, fr, es, it, pt, th).
- [ ] **Ответы ИИ на выбранном языке:**
  - [ ] Чат (AI-тренер)
  - [ ] Питание (анализ фото еды)
  - [ ] Оркестратор (дневное решение)
  - [ ] Аналитика (insight по графикам)
- [ ] **Пуш-уведомления** — sleep reminder, recovery, daily decision приходят на выбранном языке (при наличии записей для этой локали на бэкенде).
